### 🛠️ Core Stack & Engine
* **Visualização:** `Three.js R160` com suporte a `BufferGeometry`.
* **Formatos Suportados:** Decodificação binária de `.mesh` e parser `.rbxm`.
* **Interface:** Sistema de inputs reativos com **UICorner** via CSS avançado.
* **Exportação:** Módulos de unificação de malhas para `OBJ` e `GLTF`.
