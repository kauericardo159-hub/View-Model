### 🌐 Visão Geral do Projeto
A **Universal Mesh Engine PRO** é um ecossistema de renderização e edição 3D focado na decodificação e reconstrução de arquivos de malha complexos (como `.mesh` e `.rbxm`). 

Atualmente, o projeto encontra-se em **modo de hibernação técnica** enquanto a equipe trabalha na otimização dos sistemas de conversão e na correção de falhas de buffer identificadas no motor de visualização.
